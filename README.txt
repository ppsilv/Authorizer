The directory libhttpserver is used to built the libraries to compile web_service.
The directory pugixml is used to built the libraries to compile xml_handler.
You must get the necessary libs:

Install httmmicrohttp
sudo apt install httpmicrohttp

libhttpserver:
git clone https://github.com/etr/libhttpserver.git

pugixml:
git clone https://github.com/ppsilv/pugixml.git
